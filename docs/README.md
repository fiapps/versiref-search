# versiref-search Documentation

versiref-search indexes Markdown documents into SQLite databases with Bible reference and full-text search.

## For Database Consumers

If you have a versiref-search database and want to query it—by Bible reference, text string, or both—see **[Searching Databases](searching.md)**.
This covers the `search`, `context`, and `info` CLI commands, output formats, and the Python search API.

## For Database Producers

If you need to build databases from Markdown source documents, see **[Building Databases](indexing.md)**.
This covers the `index` CLI command, metadata files, config file options, and the Python indexing API.
